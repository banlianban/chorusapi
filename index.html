API 文档

GET /health

描述: 健康检查端点，返回API状态和版本信息

响应示例:

{
  "status": "healthy",
  "version": "1.0.0",
  "uptime": "running"
}

POST /extract-chorus

描述: 从上传的音频文件中提取副歌部分

参数:

  • file (必需): 音频文件 (支持 mp3, wav, m4a, flac)
  • duration (可选): 副歌时长，10-120秒，默认30秒
  • quality (可选): 质量设置 - "low", "medium", "high"，默认"high"

响应示例:

{
  "success": true,
  "chorus_start_sec": 45.2,
  "output_file_path": "outputs/abc123_chorus.wav",
  "message": "Chorus extracted successfully",
  "file_id": "abc123"
}

GET /download/{file_id}

描述: 下载提取的副歌文件

参数:

  • file_id (必需): 从extract-chorus端点返回的文件ID

DELETE /cleanup/{file_id}

描述: 清理与特定file_id关联的临时文件

GET /supported-formats

描述: 获取支持的音频格式和限制信息

响应示例:

{
  "supported_formats": [".mp3", ".wav", ".m4a", ".flac"],
  "max_file_size": "50MB",
  "max_duration": "120 seconds"
}

在线演示

🚀 快速开始

要使用在线演示功能，请先启动Chorus API服务：

方法1: 直接运行

python main.py

方法2: 使用uvicorn

uvicorn main:app --host 0.0.0.0 --port 8000 --reload

💡 服务启动后，API将在 http://localhost:8000 运行，然后就可以使用下面的演示功能了！

🛠️ 简化启动方式

也可以使用启动脚本：

python start_api.py

启动脚本会自动检查依赖、创建必要目录并启动服务
